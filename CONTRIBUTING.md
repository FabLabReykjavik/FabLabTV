# Contributing to Fab Lab TV

Thank you for your interest in Fab Lab TV.

Fab Lab TV is currently maintained by the original author, and the project is still finding its shape. Community feedback is very welcome, especially from labs that install and use the system in real spaces.

## How to help right now

The best way to contribute at this stage is to open an issue with:

- bug reports
- installation problems
- feature ideas
- screenshots of your lab using Fab Lab TV
- language corrections
- suggestions for Global Pulse sources
- ideas for making the Remote screen easier for non-technical staff

## Pull requests

Small fixes are welcome.

For larger changes, please open an issue first so we can discuss the direction before you spend time implementing something.

Good first contributions include:

- documentation improvements
- typo fixes
- language label corrections
- small UI polish
- accessibility improvements

## Project goals

Fab Lab TV should remain:

- easy for non-technical staff
- useful for real Fab Labs
- professional on a public TV screen
- simple to install
- maintainable
- friendly to the global Fab Lab community

Please keep those goals in mind when suggesting changes.
